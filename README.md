## 資料夾結構

- `papers/` — 相關論文 PDF(arXiv)
- `notes/` — 研究筆記:文獻調查(A)、根因精讀(B1)、資料集(C)、benchmark(D)、賣點(E1)、會議記錄、待辦
- `slides/` — 簡報與口頭稿
- `paper_figures/` — 論文原圖(取自 arXiv HTML 版,引用需標註原文)
- `paperskill.md` — research-idea-review skill 原始檔(已安裝於 `.claude/skills/`)
