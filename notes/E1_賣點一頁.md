## 一句話賣點
> **我們把「4D 證據一致性」（光流／scene-flow）當成一個連續、可自動驗證的 GRPO reward，去診斷並修補 VLM 在動態場景中「感知不足」的根因——這是首個把運動證據從「壓縮 prior」升級為「訓練訊號」的做法。**

一句更短的版本（放標題）：
> **Reward what the model should see: 用 4D 證據一致性當可驗證 reward,活化 VLM 的時空感知。**

---

## 支撐論據（診斷 → 方法 → 可行性 → 新穎性）

### 1. 診斷：問題真的存在,且偏「感知端」(來自 B1 / D)
- VLM 在動態時空任務與人類差距巨大:**VLM4D 人類 98.8% vs 最佳模型 62.0%(36 分差)**。
- 根因**依任務型態而異**:數學/邏輯題偏「推理未發揮」,而 **4D/時空題偏「感知不足」**——弱在「整合多重視覺線索 + 維持時間一致性」(VLM4D 診斷)。
- 現有 VLM 多靠「**聚合 2D 特徵跨時間**」,缺乏真正的運動/深度理解 → 這正是可注入 4D 證據的縫隙。

### 2. 方法:先歸因、再對症下 reward(骨架來自 MoCA,升級判準)
- 沿用 **MoCA(Bad Seeing or Bad Thinking)** 的敘事與模態信用分配:先判「壞在看 vs 壞在想」,再對感知 token 做 GRPO 信用分配。
- **我們的升級**:把感知判準從 MoCA 的「文字充分性(蒙眼推理器)」換成 **「4D 證據一致性」**——模型的時空推理必須與光流／scene-flow 證據一致才得高 reward。
- 連續化:借 **Smooth Operator / AP-GRPO** 的「Sigmoid 稠密可驗證 reward」,把 near-miss 誤差轉成稠密訊號,解決 RLVR 的稀疏痛點(A3)。

### 3. 可行性:reward 可規模化計算(來自 C)
- **不需人工 GT**:用 **Depth Anything V2 + RAFT/FlowSeek** 在任意真實影片上偽標 depth/flow → reward 可在大規模真實影片上算。
- **先驗證再泛化**:先用 **Spring / Virtual KITTI 2 / Dynamic Replica**(有 GT flow/scene-flow)驗證概念,再用 **ARKitScenes / VSI-Bench**(真實+深度)測泛化。
- 評測:**VLM4D + STI-Bench**(STI 有可量化的速度/距離題,最能展示「連續 reward」優勢)。

### 4. 新穎性:站在空白處(來自 A2 撞題檢查)
- **depth-as-reward 已擁擠**(DepthLM、Smooth Operator、3D-R1、MetaSpatial…)→ 不單賣深度。
- **flow / scene-flow 一致性當「VLM 推理的可驗證 reward」= 幾乎空白** → 這是主打新穎點。
- **Flow4Agent 只把光流當 training-free 的壓縮 prior**(選 frame/剪 token),沒當 reward → 我們把它升級為訓練訊號。

## 與最近鄰的區隔（審稿人會問的「這跟 X 有什麼不同？」）

| 最近鄰 | 它做什麼 | 我們的區隔 |
|---|---|---|
| **Flow4Agent** (2510.05836) | 光流當 **training-free 壓縮 prior**(選 frame、剪 token) | 我們把光流當 **訓練 reward**,直接優化推理與運動證據的一致性 |
| **MoCA / Bad Seeing or Bad Thinking** (2605.14054) | 感知判準 = **文字充分性**(蒙眼純文字 reasoner) | 判準 = **4D 證據一致性**,對動態/幾何任務更客觀、更適合連續 reward |
| **Smooth Operator / AP-GRPO** (2601.07695) | **深度 / 3D 物理約束**當連續可驗證 reward | 訊號換成 **光流 / scene-flow(動態運動)**,主打時間一致性而非靜態幾何 |
| **VGGRPO** (2603.26599) | latent GRPO + 相機平滑 + 幾何重投影一致性,做 **影片生成** | 我們做 **VLM 推理**(答題/推理接地),非生成;reward 服務於「想對」而非「畫對」 |
| **Video-R2** (2511.23478) | GRPO + **時間戳對齊 reward(TAR)** | 我們用 **運動場一致性** 而非時間戳對齊;訊號來自像素級 4D 證據 |
| **DepthLM / 3D-R1 / N3D-VLM** | **靜態深度 / 3D 接地** reward | 我們聚焦 **動態(motion / temporal)**,補其靜態盲區 |

## 一頁邏輯鏈（可直接做成 intro 一張投影片）
**問題**：VLM 在 4D 時空任務差 36 分,根因是「感知不足 → 整合線索與時間一致性差」
　↓
**洞察**：depth reward 已擁擠,但 **flow/scene-flow 一致性當推理 reward 是空白**
　↓
**方法**：MoCA 式歸因(壞在看/壞在想)＋ 感知判準升級為「4D 證據一致性」＋ Smooth 連續可驗證 reward
　↓
**可行**：Depth Anything V2 / RAFT 偽標 → reward 免 GT、可規模化
　↓
**驗證**：Spring/VKITTI2 驗概念 → VLM4D/STI-Bench 測能力
